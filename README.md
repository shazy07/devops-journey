# Linux Basics - Problem Solving

## Overview
This folder contains scripts and examples demonstrating Linux basics including users, groups, file permissions, and ownership.  
These are core skills for any aspiring DevOps Engineer.

## Skills Demonstrated
- Linux CLI navigation
- File system management
- Users & groups handling
- File ownership & permissions
- Basic Bash scripting

## Scripts
- `users.sh` – displays current user, user details, and groups
- `permissions.sh` – demonstrates changing file permissions
- `file_ownership.sh` – changes file ownership
